# QLS Client

## Installation

```
"repositories": [
  {
    "type": "vcs",
    "url": "https://github.com/GerardProgammeert/qls-client"
  }
],
```
```
composer require yourvendor/qls-client
```
```bash
  php artisan vendor:publish --provider="BeezMaster\QLSClient\QLSClientServiceProvider" --tag=config
```
